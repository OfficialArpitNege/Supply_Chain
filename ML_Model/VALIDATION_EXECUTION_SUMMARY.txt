2. CORRECT PREPROCESSING (Not the Issue!)
   ✓ SMOTE applied AFTER train/test split
   ✓ Training data balanced to 1:1
   ✓ Test data keeps original distribution (3.3%)
   ✓ Features engineered correctly
   ✓ Feature importance is logical

3. MODEL LEARNING PATTERN
   - With 96.7% delays in data, model learned strong pattern
   - Default prediction: "assume delay"
   - Conservative bias (appropriate for original data)
   - But creates unusable binary classification

4. NOT A POOR MODEL
   - F1-macro: 0.49 (shows poor binary performance)
   - ROC AUC: 0.993 (misleading due to imbalance)
   - Accuracy: 96.7% (just predicting majority)
   - Issue is fundamental data distribution, not model quality
